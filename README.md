# Práctica de arreglos 1
Programa que calcula la diferencia entre dos arreglos unidimensionales   y muestra el promedio de todos los datos.  

°Objetivo del proyecto.
El propósito de este ejercisio es aplicar conocimientos básicos de estructuras de datos lineales, particularmente:
*Declaración e inicialización de arreglos.
*Captura de datos por consola mediante el usuario.
*Operaciones aritméticas entre arreglos posición por posición.
*Cálculo de promedios y recorrido de estructuras.
